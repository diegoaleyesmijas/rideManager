# 🏍️ RideManager

Sistema de gestión de alquiler de motos desarrollado con PHP MVC y MySQL.

## Tecnologías
- PHP (Arquitectura MVC)
- MySQL
- Bootstrap 5
- HTML/CSS

## Funcionalidades
- Listado de motos con disponibilidad en tiempo real
- Gestión de clientes con sistema de ecoPuntos
- Registro de alquileres con validaciones de negocio
- Categorización automática de clientes (Bronce/Plata/Oro/Platino)

## Instalación
1. Clona el repositorio
2. Crea la base de datos `ridemanager` en MySQL
3. Importa `database.sql`
4. Configura `core/Database.php` con tus credenciales
5. Abre `http://localhost/ridemanager/public/index.php`
